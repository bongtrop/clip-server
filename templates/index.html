Introduction

  • Name : clip-server
  • Version : v 1.0
  • Author : Pongsakorn Sommalai
  • Github : https://github.com/bongtrop/clip-server

This project was created with relay system idea. We want some system that can only receive, send and hold data in short time. It's not secure at all!
If you want to use it, please make sure you don't put sensitive data or encrypt it already.

Getting Started back to top

This system have 4 actions save, edit, delete and get with http verb POST, PUT, DELETE and GET. The datas will store only 30s. I will example with curl.

Status code Meaning
200 Action OK
201 Clip created
404 Clip not found
406 Save clip error
409 Duplicate clip

Action get (GET)

                                    # example with curl
                                    curl http://server/your_clip_name

                                    # result
                                    # your_data_here
                                
                                    # example with request-python 
                                    r = requests.get("http://server/your_clip_name")
                                    print r.content

                                    # result
                                    # your_data_here

Action save (POST)

                                    # example with curl
                                    curl -X POST --data "your_data_here" -i http://server/your_clip_name

                                    # result
                                    # HTTP/1.0 201 CREATED
                                    # Content-Type: text/html; charset=utf-8
                                    # Content-Length: 0
                                    # Server: clip-server/v...
                                
                                    # example with request-python 
                                    r = requests.post("http://server/your_clip_name", data="your_data_here")
                                    print r.status_code

                                    # result
                                    # 201

Action edit (PUT)

                                    # example with curl
                                    curl -X PUT --data "your_data_here" -i http://server/your_clip_name

                                    # result
                                    # HTTP/1.0 200 OK
                                    # Content-Type: text/html; charset=utf-8
                                    # Content-Length: 0
                                    # Server: clip-server/v...
                                
                                    # example with request-python 
                                    r = requests.put("http://server/your_clip_name", data="your_data_here")
                                    print r.status_code

                                    # result
                                    # 200

Action delete (DELETE)

                                    # example with curl
                                    curl -X DELETE http://server/your_clip_name

                                    # result
                                    # HTTP/1.0 200 OK
                                    # Content-Type: text/html; charset=utf-8
                                    # Content-Length: 0
                                    # Server: clip-server/v...
                                
                                    # example with request-python 
                                    r = requests.delete("http://server/your_clip_name")
                                    print r.status_code

                                    # result
                                    # 200
